## 제품 개요

**Baskit** — Siri 음성 입력, AI 기반 실제 상품 링크 추천, 구매 이력 학습, 공유 리스트를 결합한 스마트 장보기 앱. "기억 → 선택 → 구매"를 원스톱으로 해결하는 것을 목표로 한다.

### 1-2. TAM (Total Addressable Market) — 전체 가용 시장

**정의**: 스마트폰을 사용하며 온라인/모바일 쇼핑을 하는 한국 소비자 전체

- 한국 모바일 쇼핑 연간 거래액: 약 198조 원
- 이 중 식품/생활용품 비중(약 25% 추정): **약 49조 원**
- **TAM = 약 49조 원** (국내 모바일 식품·생활용품 쇼핑 시장 전체)

### 1-3. SAM (Serviceable Addressable Market) — 실질 공략 가능 시장

**정의**: Baskit의 핵심 기능(음성 입력, 공유 리스트, 상품 추천)에 실질적 니즈가 있는 타겟 세그먼트

- 1인 가구 (804만 가구 중 앱 활용 가능 20~40대 비중 약 35%): 약 281만 가구
- 커플·부부 중 공동 장보기를 하는 세대 (전체 가구의 약 20%): 약 240만 가구
- 쇼핑 효율화에 관심 있는 적극 소비층 (모바일 쇼핑 MAU의 약 15%): 별도 중복 포함

합산 잠재 사용자: **약 400~500만 명**

- 1인당 연간 식품·생활용품 모바일 지출 평균: 약 80만 원 (연간 소비지출 168만 9천 원의 약 47%)
- **SAM = 약 3.2조 ~ 4조 원** (쇼핑 지출 연동 시장 기준)

### 1-4. SOM (Serviceable Obtainable Market) — 실제 점유 가능 시장

**정의**: MVP 출시 후 2~3년 내 현실적으로 확보 가능한 규모 (1인 개발, 바이럴 중심)

- 목표 MAU (3년 내): 30만 명
- 유료 구독 전환율 목표: 10%
- 유료 구독자: 3만 명 × 월 3,900원 × 12개월 = **약 14억 원 / 연**
- 어필리에이트 수익 (무료 유저 포함 거래 연동): 월 거래 유발액 가정 1인 5만 원 × 30만 명 × 3% 수수료 = **약 54억 원 / 연**
- **SOM (수익 기준) = 약 68억 원 / 연** (3년 목표 시점)

### 1-5. 주요 시장 트렌드

1. **음성 커머스 성장**: iOS Siri, Google Assistant 기반 음성 입력 활용 앱의 사용률 증가. 글로벌 음성 쇼핑 시장이 2025년까지 연 30% 이상 성장 전망.
2. **공유 경제와 가계 관리 디지털화**: 커플·부부 단위의 가계 공유 앱 수요 증가 (가계부, 공유 리스트 앱 성장세).
3. **1인 가구 급증**: 전체 가구의 36.1%가 1인 가구로, 혼자 살수록 장보기 메모·반복 구매 패턴 관리 필요성 증가.
4. **이커머스 통합 피로**: 쿠팡, 네이버쇼핑, 마켓컬리 등 플랫폼 분산으로 "어느 쇼핑몰에서 살지" 비교 피로 증가.
5. **구독 경제 보편화**: 한국 소비자의 디지털 구독 수용성 증가 (Spotify, 유튜브 프리미엄, 넷플릭스 등의 월정액 모델 일반화).

## 2. 경쟁사 분석

### 2-1. 직접 경쟁사 (장보기/쇼핑 리스트 앱)

#### OurGroceries
- **강점**: 가족·커플 공유 리스트 기능, 멀티 플랫폼(iOS/Android/Web), 오랜 사용자 기반
- **약점**: 한국 쇼핑몰 미연동, 단순 텍스트 리스트에 그침, Siri 연동 제한적, 한국어 UX 최적화 부재
- **가격**: 무료(광고) / 프리미엄 $4.99/월

#### AnyList
- **강점**: 자동 카테고리 분류, 레시피 연동, iOS 전용 고품질 UX
- **약점**: 한국 서비스 미지원, 상품 구매 링크 없음, 공유는 가족 단위 한정
- **가격**: 무료 / AnyList Complete $11.99/년

#### Any.do (Grocery 기능)
- **강점**: 할일·캘린더·쇼핑 통합, Siri/Google Assistant 연동, 글로벌 인지도
- **약점**: 한국 쇼핑몰 미연동, 상품 추천 없음, 종합 앱의 특성상 쇼핑에 집중하지 못함
- **가격**: 무료 / 프리미엄 $5.99/월

#### Bring! Shopping List
- **강점**: 깔끔한 UI, 공유 리스트, Siri Shortcuts 지원, 다국어 지원
- **약점**: 한국 쇼핑몰 미연동, 상품 추천 기능 없음, 구매 이력 학습 부재
- **가격**: 무료 / 프리미엄 $2.49/월

### 2-2. 간접 경쟁사 (한국 이커머스 앱 내 기능)

#### 쿠팡 장바구니 / 위시리스트
- **강점**: 국내 최대 쇼핑 플랫폼 신뢰도, 빠른 배송, 자체 추천 알고리즘
- **약점**: 쿠팡 단일 플랫폼 종속, 공유 리스트 기능 없음, 음성 입력 미지원, 사전 기획 장보기 지원 부족

#### 네이버 쇼핑 / 스마트스토어 찜
- **강점**: 가격 비교 기능, 네이버페이 연동, 검색 엔진 연결
- **약점**: 공유·협업 리스트 없음, 음성 입력 없음, 반복 구매 패턴 관리 부재

#### 카카오톡 채팅 장보기 리스트 (비공식)
- 많은 커플·부부가 카카오톡 채팅창에 장볼 것을 메모하는 현실적 대안
- **약점**: 체크 기능 없음, 이력 관리 불가, 쇼핑몰 연동 없음

### 2-3. 경쟁 포지셔닝 요약

```
                    [상품 링크/구매 연동]
                           높음
                            |
    쿠팡 앱 -------- Baskit (목표 포지션)
                            |
낮음 -------- [공유/협업] ---------- 높음

    Bring!                  |

```

**Baskit의 핵심 차별화 포인트**:
1. **국내 쇼핑몰 직접 연동**: 실제 구매 가능한 상품 링크를 앱 내에서 바로 제공 (경쟁사 전무)
2. **Siri 음성 입력 + 한국어 최적화**: "Baskit에 두부 추가해줘"가 즉시 작동
3. **구매 이력 기반 개인화**: 반복 구매 패턴 학습으로 추천 정확도 향상

## 4. 성장 전략 및 확장성

### 4-1. 단기 (0~12개월) — MVP 출시 및 핵심 지표 검증

- **MVP 핵심 기능**: Siri 음성 입력 + 텍스트 리스트 + 쿠팡 파트너스 링크 추천 + 공유 리스트
- **출시 채널**: App Store 우선 (iOS 타겟 Siri 연동 강점 활용), 이후 Android 확장
- **초기 유저 확보**: 부부/커플 커뮤니티(네이버 카페, 맘카페), 1인 가구 라이프스타일 인플루언서 협업
- **핵심 지표**: MAU, D7/D30 리텐션, 음성 명령 사용률, 공유 리스트 활성 팀 수, 쿠팡 링크 클릭률

### 4-2. 중기 (12~36개월) — 유료 구독 전환 및 멀티 쇼핑몰 확장

- 네이버쇼핑 API 연동 → 가격 비교 기능 추가 (유료 전환 유인 강화)
- 구매 이력 학습 알고리즘 고도화 (반복 구매 주기 예측, 재구매 알림)
- Android 버전 출시 → MAU 2배 이상 성장 기대
- 마켓컬리, SSG 등 신선식품 특화 쇼핑몰 연동

### 4-3. 장기 (36개월 이후) — 인접 시장 확장

- **B2B 모델 탐색**: 소규모 식자재 공동구매 커뮤니티, 공동 장보기 그룹 기능
- **레시피 연동**: 요리 레시피 앱과 연동하여 "이 요리 재료 한번에 장보기" 기능
- **정기 배송 연동**: 자주 사는 상품을 정기 구독 배송으로 자동 연결 (쿠팡 로켓배송 등)
- **해외 확장**: 일본(1인 가구 비율 세계 최고), 동남아(이커머스 급성장) 시장 타겟

### 4-4. 인접 시장 확장 가능성

| 인접 시장 | 기회 | 연관성 |

## 5. 핵심 가정 및 검증 방법

### 가정 1: 사용자는 음성 입력을 실제로 사용할 것이다

- **근거**: Siri 연동 장보기 앱(Any.do, Bring!)의 Siri Shortcuts 지원 요구가 App Store 리뷰에서 반복적으로 등장. 이미 Apple Reminders로 Siri 장보기 리스트를 사용하는 습관이 존재.
- **검증 방법**: MVP 출시 후 음성 명령 사용률(%) 측정. 목표: 전체 아이템 추가 중 30% 이상이 Siri 경유
- **리스크**: 음성 입력이 실제 사용 환경(지하철, 사무실)에서 불편할 수 있음 → 텍스트 입력 UX 동등하게 유지

### 가정 2: 쿠팡 파트너스 링크 클릭 후 실구매 전환율이 충분히 높을 것이다

- **근거**: 장보기 앱은 구매 의도가 명확한 상태에서 사용 → 일반 블로그/SNS 어필리에이트(1~5%)보다 전환율 높을 것으로 예상
- **검증 방법**: 출시 후 3개월 내 링크 클릭률(CTR) 및 구매 전환율(CVR) 실측
- **목표**: CTR 25% 이상, CVR 15% 이상
- **리스크**: 전환율이 예상보다 낮을 경우 어필리에이트 수익 대폭 감소 → 구독 수익 비중 확대로 대응

### 가정 3: 멀티 쇼핑몰 연동이 유료 구독 전환의 핵심 유인이 될 것이다

- **근거**: 단일 쿠팡 링크만 제공되는 무료 티어는 유저에게 가격 비교 니즈를 유발, 멀티 연동 유료 전환 촉진
- **검증 방법**: 네이버쇼핑 연동 베타 출시 후 무료→유료 전환율 비교 A/B 테스트
- **목표**: 멀티 연동 기능 노출 시 유료 전환율 2배 이상

### 가정 4: 공유 리스트가 바이럴 성장의 핵심 채널이 될 것이다

- **근거**: 커플·부부가 공유 리스트를 사용하려면 파트너도 앱을 설치해야 함 → 자연스러운 초대 발생
- **검증 방법**: 공유 리스트 초대 링크를 통한 신규 가입 비율 측정
- **목표**: 신규 유저의 30% 이상이 초대 링크 경유

### 가정 5: 1인 개발·저비용 인프라로 Year 1 손익분기점 달성 가능

- **주요 비용 항목**:
  - 클라우드 서버 (AWS/GCP 최소 구성): 월 30~50만 원
  - 쇼핑몰 API 사용료: 월 0~20만 원 (무료 API 우선 활용)
  - 앱스토어 수수료 (구독 수익의 30%): 비례 발생
  - 마케팅 (초기 무료 채널 중심): 월 10~30만 원
- **손익분기점**: 유료 구독자 약 300명 수준에서 고정 비용 커버 가능
- **검증 방법**: 출시 6개월 시점 MRR vs. 운영 비용 비교

### 핵심 성공 요인

1. **Siri 연동 UX 완성도**: 음성 입력이 매끄럽지 않으면 핵심 차별점이 소멸. iOS SiriKit 구현 품질이 제품 성패를 좌우.
2. **쇼핑몰 API 파트너십**: 쿠팡 파트너스 외 네이버쇼핑 API, 마켓컬리 제휴 연동이 유료 전환율의 핵심.
3. **초기 유저 리텐션**: 장보기 앱 특성상 주 1~2회 반복 사용이 핵심. D30 리텐션 40% 이상 목표.
4. **공유 리스트 바이럴**: 커플·부부 초대 구조로 마케팅 비용 최소화.

### 주요 고려사항 및 리스크

1. **쇼핑몰 API 의존성**: 쿠팡·네이버 등 플랫폼이 API 정책을 변경하거나 파트너스 수수료를 조정할 경우 수익 구조에 직접 타격. 다양한 플랫폼 분산이 필요.
2. **대형 플랫폼의 유사 기능 내재화 위험**: 쿠팡이나 네이버가 공유 리스트·음성 입력 기능을 자사 앱에 추가할 경우 경쟁 심화. 이력 데이터 기반 개인화 고도화로 방어.
3. **1인 개발 리소스 제약**: 동시 다발적 기능 개발보다 핵심 루프(음성 입력 → 추천 → 구매) 완성에 집중하는 MVP 전략 필수.
4. **앱스토어 수수료**: 구독 수익의 30%가 애플에 귀속. 연 구독 전환 유도로 실질 수수료율 완화 (2년차 이후 15%로 감소).
