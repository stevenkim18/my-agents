## 제품 개요

- **제품명**: WeCal (함께 쓰는 캘린더)
- **핵심 가치**: 가족/커플을 위한 실시간 공유 캘린더 — 일정 변경 즉시 반영, 공동 일정 수락/거절 플로우 내장
- **타겟**: 부부, 연인 등 소규모 일정 공유 그룹
- **플랫폼**: iOS 우선, 1인 개발

### 시장 성장률 및 주요 트렌드

- **전체 캘린더 앱 시장**: CAGR 10.4% (2024~2032)
- **공유 캘린더 앱 세그먼트**: CAGR 12.4% (2024~2031)
- **커플/가족 특화 앱 세그먼트**: 별도 통계 부재하나 데이팅·커플 앱 카테고리 자체가 지속 성장 중

**핵심 성장 동인**:
1. **AI 연동 수요 증가**: TimeTree가 SK텔레콤과 파트너십을 통해 AI 기반 일정 제안 기능 개발 중. 일정 데이터를 자연스러운 AI 트리거로 활용하는 방향
2. **크로스 플랫폼 동기화 한계**: Google Calendar ↔ Apple Calendar 간 완전한 실시간 양방향 공유가 여전히 어려워, 전용 앱 수요가 지속됨
3. **원격·유연근무 확산**: 가족 구성원 간 일정 가시성 요구 증가
4. **커플/가족 특화 UX 수요**: 범용 캘린더 앱으로는 충족되지 않는 감성적·기능적 니즈 존재

| **Apple/Google Calendar** | 범용 캘린더 플랫폼 내장 | 설치 불필요, 방대한 생태계 연동 | 크로스 플랫폼 실시간 공유 한계, 커플/가족 UX 부재, 공동 초대/수락 플로우 없음 |

### WeCal의 차별화 포인트

1. **공동 일정 초대 + 수락/거절 플로우 네이티브 내장**: 기존 앱들은 이 플로우가 어색하거나 별도 메시지 앱을 통해야 함. WeCal은 이를 핵심 UX로 설계
2. **외부 캘린더(Google Calendar 등) 양방향 연동**: 개인 캘린더와 공유 캘린더를 하나의 앱에서 관리
3. **커플/가족에 최적화된 UX**: Cozi의 기능 과잉, TimeTree의 범용성과 달리 소규모 친밀 그룹에 집중
4. **실시간 반영**: "내가 추가하면 상대방이 즉시 본다"는 명확한 핵심 가치

## 3. 타겟 고객 분석

### 주요 고객 세그먼트

| 세그먼트 | 특징 | 우선순위 |
|---------|------|---------|
| **동거 커플 / 부부 (20~40대)** | 일상 일정 공유 빈도 높음, 맞벌이로 인해 일정 조율 필요성 극대화 | 최우선 |
| **장거리 연애 커플** | 물리적으로 떨어져 있어 디지털 일정 공유 의존도 높음 | 높음 |
| **영유아 자녀를 둔 가족** | 육아 스케줄, 학원 일정 공유 등 복잡한 일정 조율 필요 | 중간 (향후 확장) |
| **싱글 파워유저** | 개인 일정 관리 + 지인과 간헐적 공유 | 낮음 (보조 세그먼트) |

### 핵심 Pain Point

1. **"일정 바뀌었어?" 커뮤니케이션 비용**: 카톡/문자로 일정 변경을 별도 전달해야 하는 번거로움
2. **크로스 플랫폼 공유 불편**: 한 명은 Google Calendar, 한 명은 Apple Calendar를 쓸 때 실시간 공유가 사실상 불가능
3. **공동 약속 잡기의 마찰**: "그날 어때?" → "나는 괜찮아" → 이를 정식 일정으로 확정하는 과정이 분리되어 있음
4. **일정 가시성 부재**: 상대방 일정을 보기 위해 별도로 물어봐야 하는 상황

### 구매 결정 요인

1. **설치 및 연결 간소함**: 상대방도 설치해야 하는 앱 특성상, 온보딩 마찰이 낮아야 설득 가능
2. **실시간성 신뢰**: 업데이트 딜레이가 없다는 확신
3. **외부 캘린더 연동**: 기존 캘린더 데이터를 버리지 않아도 된다는 안심감
4. **푸시 알림 품질**: 중요한 일정 변경을 놓치지 않는다는 신뢰
5. **무료 시작 가능 여부**: 두 사람이 함께 써야 하는 앱 특성상 진입 장벽 낮을수록 유리

## 4. 수익화 모델 검토

### 제안 가능한 수익화 모델

수익화 모델이 현재 미정이므로, 유사 앱 사례를 바탕으로 3가지 모델을 평가합니다.

#### 모델 A: 프리미엄 구독 (Freemium + Subscription) — **추천**

| 항목 | 내용 |
|------|------|
| 무료 티어 | 공유 캘린더 기본 기능, 최대 2인, 30일 이전 일정 조회 제한 |
| 유료 티어 | 무제한 히스토리, 고급 알림 설정, 다중 캘린더 연동, 광고 제거 |
| 가격 벤치마크 | Cupla: $4.99/월 또는 $34.99/년 / Cozi Gold: $39/년 |
| 예상 단가 | 월 3,900원~5,900원 / 연 29,000원~39,000원 (한국 시장 기준) |
| 전환율 가정 | 프리미엄 전환율 2~5% (업계 중간값) |

#### 모델 B: 광고 + 선택적 업그레이드

- 장점: 초기 사용자 확보 용이
- 단점: 커플/가족 앱 특성상 광고가 UX를 해침. Cozi의 광고 정책 변경 이후 대규모 이탈 사례가 경고 신호

#### 모델 C: 일회성 구매 (Lifetime Deal)

- 장점: 초기 수익 집중, 구독 피로감 없음
- 단점: 지속적인 서버 비용 감당 어려움, 1인 개발 장기 운영 부담

### 수익성 시뮬레이션 (보수적 추정)

| 지표 | 1년차 | 2년차 | 3년차 |
|------|-------|-------|-------|
| 누적 다운로드 | 5만 | 20만 | 50만 |
| 유료 전환율 | 2% | 3% | 4% |
| 유료 사용자 수 | 1,000명 | 6,000명 | 20,000명 |
| 연 ARPU | 35,000원 | 35,000원 | 35,000원 |
| 연 매출 추정 | 3,500만 원 | 2억 1,000만 원 | 7억 원 |

> 참고: 시그널링의 경우 출시 7개월 20만 다운로드 달성. WeCal이 유사 수준의 성장을 보인다면 1년차 목표는 달성 가능한 수준.

### 추가 수익화 기회

- **커플 관련 제휴 광고** (비침습적): 레스토랑 예약, 꽃 배달, 여행 상품 등 일정과 연계된 맥락 광고
- **프리미엄 테마/스킨**: 캘린더 커스터마이징 인앱 결제
- **B2B/B2B2C 확장**: 장기적으로 소규모 팀 일정 관리 솔루션으로 확장

## 5. 비즈니스 기회 및 확장성

### 단기 성장 기회 (0~12개월)

- **Cozi 이탈 사용자 흡수**: 2024년 5월 Cozi의 유료화 정책 강화 이후 사용자 이탈이 활발. "완전 무료 시작" 포지셔닝으로 공략 가능
- **한국 시장 집중**: 시그널링의 성공 사례에서 알 수 있듯, 한국 커플 앱 시장은 로컬 앱에 친화적
- **App Store 최적화 (ASO)**: "커플 캘린더", "공유 일정" 키워드 경쟁이 TimeTree 등 대형 앱 위주로 형성 — 틈새 키워드 공략 가능

### 중기 성장 기회 (1~3년)

- **Android 확장**: iOS 검증 후 Android 출시로 TAM 2~3배 확대
- **글로벌 출시**: 일본(TimeTree 발원지 시장으로 경쟁 치열하나 수요 검증됨), 동남아 시장 진출
- **AI 기능 통합**: "이번 주 함께 할 수 있는 시간 찾아줘" 같은 AI 기반 일정 제안. TimeTree가 이미 이 방향으로 투자 중으로, 선점 기회 존재

### 장기 성장 기회 (3년 이상)

- **커플/가족 라이프스타일 허브**: 공유 가계부, 여행 계획, To-do 등으로 확장하여 Cozi의 포지션 대체
- **소규모 팀 버전**: 스타트업, 프리랜서 팀 등 2~5인 소규모 그룹 일정 관리로 B2B 확장
- **데이터 기반 인사이트**: 익명화된 일정 패턴 데이터를 기반으로 커플/가족 대상 광고/제휴 플랫폼화

### 인접 시장 확장 가능성

| 인접 시장 | 연계성 | 진입 난이도 |
|-----------|--------|-------------|
| 커플 다이어리/기록 앱 | 높음 (일정 데이터 자연 연결) | 낮음 |
| 공유 가계부 | 높음 (함께 쓰는 앱의 연장선) | 중간 |
| 소규모 팀 일정 관리 | 중간 | 중간 |
| 이벤트/약속 플랫폼 | 중간 | 높음 |

## 6. 종합 의견

### 사업 타당성 평가: **중-상 (Medium-High)**

### 평가 근거

**긍정 요인**:
- 글로벌 공유 캘린더 앱 시장이 CAGR 12.4%로 성장 중인 구조적 성장 시장
- Cozi의 유료화 정책 변경으로 발생한 시장 공백이 실재함
- 한국 시장에서 시그널링의 빠른 성장이 로컬 커플 앱의 수요를 검증
- Google/Apple Calendar의 크로스 플랫폼 한계는 전용 앱의 존재 이유를 지속적으로 만들어 줌
- 1인 개발 + iOS 우선으로 초기 비용 최소화 가능

**제약 요인**:
- TimeTree(7,000만 사용자), Cozi, Cupla 등 기존 강자들의 네트워크 효과가 강함
- "두 사람 모두 설치해야 하는" 앱 특성상 바이럴 계수(K-factor) 확보가 핵심 과제
- 수익화 모델 미정 — 초기부터 수익화 설계를 포함한 제품 로드맵 필요
- 1인 개발 체제로 기술 부채 및 운영 리스크 존재

### 핵심 성공 요인

1. **온보딩 마찰 최소화**: 상대방을 초대하고 함께 사용하기까지의 과정이 2~3단계 이내여야 함
2. **실시간 동기화 안정성**: 핵심 가치 명제(실시간 반영)가 기술적으로 확실히 작동해야 신뢰 확보 가능
3. **외부 캘린더 연동 완성도**: Google Calendar 연동이 끊기거나 중복 이벤트가 발생하면 즉각 이탈
4. **초기 사용자 리텐션 전략**: 커플/가족 앱은 이탈 시 두 사람 모두 이탈하는 구조. 습관 형성을 위한 푸시 알림 최적화 필수
5. **명확한 수익화 모델 조기 확립**: 무료로 사용자를 모은 뒤 유료화할 경우 Cozi처럼 반발 위험. 처음부터 프리미엄 티어를 명시하고 가치를 입증하는 방식 권장

### 주요 고려사항

1. **MVP 범위 제어**: 1인 개발로 외부 캘린더 연동 + 실시간 공유 + 푸시 알림을 동시에 완성도 높게 구현하는 것은 도전적. 우선순위를 명확히 하여 "딱 이것만큼은 완벽하게" 전략 권장
2. **플랫폼 의존 리스크**: Google Calendar API 정책 변경 또는 Apple 정책 변화에 따른 연동 중단 리스크 사전 검토 필요
3. **수익화 모델 결정 시기**: 사용자 10,000명 이전에 수익화 모델을 확정하고 테스트해야 스케일업 시 혼란 방지
4. **경쟁사 모니터링**: TimeTree의 AI 기능 고도화 속도를 주시하며 차별화 포인트 유지 전략 필요
